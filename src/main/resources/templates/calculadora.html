Calculadora científica

Indicaciones

  • Usa los botones para construir la expresión y presiona = para calcular.
  • Las funciones trigonométricas trabajan en grados: sin(90) = 1.
  • Para potencias escribe la base, presiona x^n y el exponente: 2**8 = 256.
  • Para factorial escribe el número entre paréntesis: factorial(5) = 120.
  • Presiona C para limpiar y ⌫ para borrar el último carácter.
